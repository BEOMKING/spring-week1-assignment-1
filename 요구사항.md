- 공통
  - [ ] URI를 잘못 입력한 경우 에러메시지 반환
- ToDo 목록 얻기 - `GET /tasks`  
- ToDo 상세 조회하기 - `GET /tasks/{id}`  
- ToDo 생성하기 - `POST /tasks`  
  - [ ] JSon 으로 넘어오는 값 파싱
    
    [{,     "title":"BJP", }]
    [{,     "abc": 1,,     "title":"BJP", }]
- ToDo 제목 수정하기 - `PUT/PATCH /tasks/{id}`  
- ToDo 삭제하기 - `DELETE /tasks/{id}`
